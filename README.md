# Island Runner Adventure Game

A console-based adventure game developed in Java where players navigate through an island, make decisions, overcome challenges, and progress through different stages of the game.

## Features
- Interactive text-based gameplay
- Decision-based progression
- User input handling
- Modular programming
- Object-Oriented Programming concepts

## Technologies Used
- Java
- OOP
- Scanner Class
- Conditional Statements
- Loops

## Learning Outcomes
This project helped strengthen my understanding of Java fundamentals, problem-solving, modular programming, and application development.
